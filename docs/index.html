✨ SPAWN Animation - Unique Feature

Apa itu SPAWN?

SPAWN adalah sistem animasi unik MCLANG yang membuat animasi kompleks menjadi sangat mudah. Dengan syntax sederhana, Anda bisa membuat animasi smooth dan professional dengan stagger otomatis.

Preset Animations

Preset Deskripsi Use Case
SpawnPresets.fadeIn() Fade in dari transparan Entrance animations
SpawnPresets.slideInLeft() Slide dari kiri dengan fade List item animations
SpawnPresets.slideInUp() Slide dari bawah dengan fade Card reveals
SpawnPresets.scaleIn() Scale dari kecil dengan fade Modal opens
SpawnPresets.bounceIn() Bounce dengan spring easing Playful animations

Simple SPAWN Example

Fade In Animation

Buat elemen fade in dengan 1 baris kode:

element FadeInBox { const box = useRef() useEffect(() => { spawn(box, SpawnPresets.fadeIn(600)) }, []) <div ref={box}> Elemen ini fade in dengan smooth </div> }

Staggered List Animation

Animate Multiple Items

Animasi list items dengan stagger otomatis:

element StaggeredList { const [items, setItems] = useState([1, 2, 3, 4, 5]) const listRef = useRef() useEffect(() => { const elements = listRef.current?.querySelectorAll('.item') if (elements) { spawnStagger(Array.from(elements), { duration: 500, stagger: 100, // 100ms delay antar item from: { opacity: 0, translateY: 20 }, to: { opacity: 1, translateY: 0 } }) } }, [items]) <div ref={listRef} class="list"> {items.map(item => ( <div key={item} class="item"> Item {item} </div> ))} </div> style { .list { display: flex; flex-direction: column; gap: 1rem; } .item { padding: 1rem; background: linear-gradient(135deg, #ffe4e9 0%, #fff5f7 100%); border-radius: 8px; } } }
