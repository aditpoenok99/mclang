# 🎨 Design & Branding

## Color Palette

MCLANG menggunakan warna-warna pink dan pastel yang nyaman di mata untuk pengalaman coding jangka panjang.

### Primary Colors

```
🌸 Primary Pink: #ff69b4
💜 Deep Purple: #d946ef
🎀 Light Pink: #ff85c1
💕 Pastel Pink: #ffc0d0
```

### Background Colors

```
✨ Light Background: #fff5f7
☁️  Lighter Background: #fffafc
🌺 Soft Pastel: #ffe4e9
```

### Text Colors

```
🖤 Dark Text: #6b3f7f
💭 Light Text: #9b6ba8
```

## Why Pink & Pastel?

✅ **Eye-Friendly** - Mengurangi strain pada mata
✅ **Modern Aesthetic** - Terlihat contemporary dan menarik
✅ **Memorable** - Membuat MCLANG mudah diingat
✅ **Inclusive** - Nyaman untuk berbagai preferensi visual
✅ **Professional** - Tetap terlihat profesional dan serious

## Design Philosophy

1. **Simplicity** - Keep interface clean and minimal
2. **Accessibility** - Ensure colors pass WCAG contrast requirements
3. **Consistency** - Use same palette across all platforms
4. **Beauty** - Make coding a pleasant experience

## Usage Guidelines

### For Gradients

```css
background: linear-gradient(135deg, #ff69b4 0%, #d946ef 100%);
```

### For Borders

```css
border: 2px solid #ffd1dc;
```

### For Hover Effects

```css
background: linear-gradient(135deg, #ff85c1 0%, #d946ef 100%);
transition: all 0.3s ease;
```

## Accessibility

Semua warna telah ditest untuk memastikan:
- ✅ Contrast ratio >= 4.5:1 untuk teks
- ✅ Tidak hanya bergantung pada warna
- ✅ Readable untuk colorblind users
- ✅ Comfortable untuk extended viewing

Untuk informasi lebih lanjut, kunjungi docs/index.html
