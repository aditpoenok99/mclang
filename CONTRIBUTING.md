# Contributing to MCLANG

Terima kasih telah tertarik untuk berkontribusi pada MCLANG! Panduan ini akan membantu Anda memulai.

## Code of Conduct

Proyek ini berfungsi di bawah Code of Conduct yang inklusif. Dengan berpartisipasi, Anda diharapkan untuk mematuhi kode ini.

## How Can I Contribute?

### Reporting Bugs

Sebelum membuat laporan bug, cek daftar issue karena Anda mungkin menemukan bug yang sudah dilaporkan. Jika Anda menemukan bug yang baru:

- **Gunakan judul deskriptif**
- **Berikan contoh spesifik untuk mendemonstrasikan langkah-langkah**
- **Jelaskan perilaku yang diamati dan apa yang Anda harapkan**
- **Sertakan screenshots jika memungkinkan**

### Mengusulkan Enhancement

Pengembangan enhancement diajukan sebagai issue dengan deskripsi yang jelas tentang use case yang dimaksud.

### Pull Requests

Setelah fork dan membuat branch fitur Anda:

1. Commit perubahan Anda dengan pesan yang jelas
2. Push ke branch Anda
3. Buat Pull Request dengan deskripsi yang detail

## Development Setup

```bash
git clone https://github.com/aditpoenok99/mclang.git
cd mclang
npm install
npm run build
```

## Testing

```bash
npm test
```

## Coding Guidelines

- Gunakan TypeScript untuk semua kode baru
- Follow ESLint rules
- Tambahkan unit tests untuk fitur baru
- Update dokumentasi
